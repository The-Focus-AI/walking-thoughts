# Build v1 as an Android-first PWA

Version 1 will be an installable PWA targeting Chrome on the user's Pixel 9. It will store Captures and mixed media locally, work without connectivity, and guarantee that synchronization resumes when the app is open and online; background synchronization while the app is closed is only a best-effort enhancement. This favors rapid web development and installation simplicity, with a native Android wrapper reserved for limitations discovered through real-device testing.
