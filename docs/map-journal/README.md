# Map Journal (#14)

The Map Journal makes the Offline Region the primary review surface:
media-aware, clustered Capture markers over the trail-first topographic map,
compact Capture previews opening complete Thread context (adjacent panel on
desktop, bottom sheet on mobile), and follow-up Captures through the ordinary
Capture and Enrichment pipeline.

Work in progress. The implementation follows the reviewing-ui prototype
verdict and rewrites rather than promotes the throwaway prototype.
